# 方程求解器功能文档

**版本:** v2.1  
**日期:** 2026-03-11

---

## 📐 功能概述

方程求解器是 iOS 计算器的扩展功能，支持用户通过输入系数来求解代数方程。当前版本支持两种类型的方程：

### 支持的方程类型

1. **线性方程 (Linear Equation)**
   - 格式：`ax + b = c` 或简化版 `ax = b`
   - 示例：`2x + 3 = 7` → 解为 `x = 2`

2. **二次方程 (Quadratic Equation)**
   - 格式：`ax² + bx + c = 0`
   - 示例：`x² - 5x + 6 = 0` → 解为 `x₁ = 2, x₂ = 3`

---

## 🎯 使用场景

### 线性方程适用场景:
- **商业计算**: 成本定价、利润分析
  - 例：每件商品成本$10，售价$15，目标利润$100 → `5x = 100` → `x = 20`件
  
- **简单比例问题**: 
  - 例：3 小时完成 60% 工作，需要多少小时完成？→ `0.6/3 = 1/x`

### 二次方程适用场景:
- **面积计算**: 已知矩形面积和长宽关系
  - 例：面积 24㎡，长比宽大 2m → `x(x+2) = 24` → `x² + 2x - 24 = 0`

- **物理问题**: 抛体运动、自由落体
  - 例：物体下落距离 s = ½gt²，已知 s 求 t

## 🔧 技术实现

### 核心类: `EquationSolver`

#### 线性方程求解算法:
```
ax + b = c
→ x = (c - b) / a

特殊情况:
- a = 0, b = c → 无穷多解
- a = 0, b ≠ c → 无解
```

#### 二次方程求解算法 (求根公式):
```
ax² + bx + c = 0

判别式 Δ = b² - 4ac

Δ > 0: 两个不同实根
  x₁ = (-b + √Δ) / 2a
  x₂ = (-b - √Δ) / 2a

Δ = 0: 一个重根
  x = -b / 2a

Δ < 0: 无实根 (复数解暂不支持)
```

### 精度处理:
- 使用 `Decimal` 类型保证计算精度
- 结果最多显示 6 位小数，自动去除末尾零

## 📱 UI/UX 设计

### 界面布局:
1. **顶部**: 方程类型切换按钮 (线性 ↔ 二次)
2. **中部**: 系数输入框 (a, b, c)
3. **底部**: 
   - "求解"按钮
   - "清除"按钮
   - 结果显示区域

### 交互流程:
1. 选择方程类型
2. 输入系数 a, b, c
3. 点击"求解"或按回车键
4. 查看结果描述和解的值

## 📊 测试用例

### 线性方程:
| 输入 | 预期结果 |
|------|----------|
| a=2, b=3, c=7 | x = 2 |
| a=5, b=0, c=100 | x = 20 |
| a=0, b=5, c=5 | 无穷多解 |
| a=0, b=5, c=10 | 无解 |

### 二次方程:
| 输入 | 预期结果 |
|------|----------|
| a=1, b=-5, c=6 | x₁ = 2, x₂ = 3 |
| a=1, b=-4, c=4 | x = 2 (重根) |
| a=1, b=0, c=1 | 无实根 (Δ < 0) |

## 💡 使用技巧

1. **快速切换**: 点击顶部按钮可在两种方程类型间切换
2. **回车求解**: 在输入框中按回车键可直接求解，无需点击按钮
3. **清除重置**: "清除"按钮会清空所有输入和结果
4. **精度控制**: 结果自动去除末尾零，保持简洁显示

---

**备注:** 方程求解器作为独立 Tab 存在，与主计算器功能互补，适合需要频繁进行代数计算的用户。
