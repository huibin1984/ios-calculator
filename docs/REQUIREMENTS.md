# 需求文档

## 1. 功能需求

### 1.1 基础运算 (FR-001)
- **描述**: 支持加、减、乘、除四则运算
- **优先级**: P0 (必须)
- **验收标准**: 
  - 输入 "2 + 3 =" 显示结果 5
  - 连续运算正确（如 2 + 3 + 4 = 9）

### 1.2 科学计算 (FR-002)
- **描述**: 提供高级数学函数
- **优先级**: P0
- **子功能**:
  - 三角函数：sin, cos, tan
  - 对数：log (以 10 为底), ln (自然对数)
  - 幂运算：x², x³, xʸ
  - 开方：√x, ∛x
  - 常数：π, e

### 1.3 语音反馈 (FR-003)
- **描述**: 按键时朗读内容，帮助商务人士确认输入
- **优先级**: P0
- **验收标准**:
  - 按下数字键 "5" → 朗读 "五" 或 "Five"
  - 按下操作符 "+" → 朗读 "加" 或 "Plus"
  - 语音延迟 < 200ms
  - 可手动开关语音功能

### 1.4 百分比计算 (FR-004)
- **描述**: 支持百分比运算（商务场景常用）
- **优先级**: P1
- **示例**: 
  - 50 + 10% = 55
  - 200 × 15% = 30

### 1.5 正负号切换 (FR-005)
- **描述**: 一键切换当前数字的正负
- **优先级**: P1

### 1.6 清除功能 (FR-006)
- **描述**: 提供两种清除方式
- **子功能**:
  - C: 清除当前输入
  - AC: 全部清除，重置计算器

## 2. 非功能需求

### 2.1 性能 (NFR-001)
- 计算响应时间 < 50ms
- 语音反馈延迟 < 200ms
- 应用启动时间 < 2s (冷启动)

### 2.2 可用性 (NFR-002)
- 按钮尺寸 ≥ 60pt (适合误触)
- 支持单手操作（常用功能在底部）
- 高对比度配色，便于阅读

### 2.3 兼容性 (NFR-003)
- 最低支持 iOS 16.0
- 适配 iPhone SE 到 Pro Max 所有尺寸
- 支持横竖屏切换

### 2.4 离线能力 (NFR-004)
- 无需网络连接即可完整使用
- 语音功能不依赖网络 TTS

## 3. UI/UX 需求

### 3.1 布局
- 显示屏：顶部，显示当前输入和结果
- 按钮区：7 列网格（科学模式）或 4 列网格（基础模式）
- 语音开关：右上角明显位置

### 3.2 交互
- 点击按钮有视觉反馈（缩放/颜色变化）
- 语音朗读与视觉反馈同步
- 支持长按快速输入（可选扩展）
